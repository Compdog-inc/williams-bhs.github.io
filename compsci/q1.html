Things to know after Q1

Expand the topics below to review the things that you should know how to do from scratch off the top of your head. Knowing these things will make learning new things much easier. Not knowing these things will make everything harder.

The fundamental basic HTML template.

<!DOCTYPE html> 
<html>
  <head>
    <title>A title</title>
  </head>  
  <body>
    <div>      
      <h1>Header</h1>
      <p>
        Here is a paragraph.
      </p>
    </div>
  </body>
</html>

Link to a CSS stylesheet with:

<!DOCTYPE html> 
  <html>
    <head>
      <title>A title</title>
      <link rel="stylesheet" href="mystyle.css"> 
    </head>  
    <body>
      ...

Here are some basic stylings! 

body {
  background-color: #ddd; 
  font-family: 'Courier New', monospace; 
}

h1 {
  color: #dcc; 
}

p {
  background-color: #fff; 
  border: 1px solid black; 
  border-radius: 7px; 
  width: 600 px; 
  margin: auto; 
}

CSS classes are a great way to modify any number of elements at once. 

<img class="profile-pic" src="picture.png">
    <div class="bio"> ... </div>

The classes above can be styled with 

.profile-pic{
  width: 150px; 
  border-radius: 50%; 
}
.bio {
  margin: 15px; 
  padding: 10px; 
  font-style: italic;  
}

HTML Ids are uniuqe to an element. For example, there might be one primary container: 

<div id="body-container"> ...

Which you can style with 

#body-container {
  width: 800px; 
  margin: auto; 
  background-color: #fff; 
}

Note that classes use the dot selector and ids use the # selector.

You connect an HTML document to a JavaScript file with:

<script src="filename.js"></script>

Given the following HTML elements: 

<input type="text" id="input-box"></input>
<div id="display-div"></div>

We can get the text in the textbox and udpate the div with: 

let input_box = document.getElementById("input-box");
let output_div = document.getElementById("display-div"); 

let text = input_box.value; 
let message = "You entered: " + text; 
output_div.innerHTML = message;

You know from the very first lesson in CodeCademy JavaScript, that three ways to declare variables include:

const username = "Jed"; 
var num = 15.2; 
let output_div = document.getElementById("screen");

In these cases, username, num, and output_div are all variable names with different values. It is also useful to be able to get input from a user. It is a good idea to do this through a text box, but we can also use our familiar prompt:

A JavaScript function requires the 'function' keyword, a function name, and parentheses around any parameters the function takes as input. Here are several examples: 

function triple(num) {
  return 3*num; 
}

function absolute_value(x) {
  if (x < 0) {
    return -x; 
  }
  return x; 
}

function update_div_from_text() {
  let input_box = document.getElementById("input-box");
  let output_div = document.getElementById("display-div"); 
  
  let text = input_box.value; 
  let message = "You entered: " + text; 
  output_div.innerHTML = message;  
}
