Computing and Data Science

Welcome to Computing and Data Science

This course introduces students to a broad set of foundational skills in applied modeling.

A core philosophy of this course is that fundamentals are important. It is easier than ever to open a Python notebook and hit "Run" on cell after cell until you have a trained neural network to detect cats. Actually knowing things is going to be an increasingly valuable asset, and my goal is to help you actually learn fundamentals so that you can apply, create, and understand.
